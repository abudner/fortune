# Fortunka

Projekt na laboratoria z Architektury serwisów internetowych .

## Funkcjonalość

Serwis pozwala użytkownikom tworzenie,edytowanie,wyszukiwanie i 
komentowanie cytatów. 

### Używane gemy 
* gem 'omniauth','~>0.3.2' 
* gem 'will_paginate', '3.0.pre2' 
* gem 'kaminari'
* gem 'acts_as_commentable'
* gem 'responders'
